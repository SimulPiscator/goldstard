Hardware

  • Goldstar FFH-212 micro-hifi system, ca. 1992
    audio controls through a TDA7318 integrated amplifier circuit
    5V microcontroller power available in stand-by mode
    • implemented through 7805 linear voltage regulator
    • can deliver up to 1.5A with appropriate heat sink
  • Raspberry Pi model B
    ARMv6 running at 850MHz
    496MB RAM available
    integrated 11bit PWM stereo audio
    Edimax low-power Wi-Fi dongle
  • RPi integration
    • physically replaces tape deck mechanism, accessible through tape lid
    • powered from stand-by power
    • controls audio through I2C connection to TDA7318
    • controls power and CD through IR LED mounted above remote sensor
      Linux IR Control Daemon (lircd)
  • MPlayer2 in slave mode plays audio from network
  • Audio quality
    FFH-212 CD player dynamic range specified as 68dB = 11bit, matching RPi PWM output
    audio power specified as 2x15W RMS at 10% THD, acceptable for video playback purpose

GUI control interface

  • Served as an interactive web page
  • Written in C++ using Wt web application library
  • Requires a HTML5 browser with JavaScript enabled
  • Designed to fit a smart phone screen
  • Allows multiple synchronous control sessions

HTTP control interface

  • /state
    Output is plain text listing all state variables and their values.
  • /control?Power=0
    /control?Power=1
    Switch power on or off.
    Cannot be combined with control of other state variables.
    Power state change requires a few seconds to complete, and no control requests are accepted during that time.
  • /control?Source=CD&GainCD=0&VolumeL=-36&VolumeR=-36
    Changes state variables according to parameters.
    Parameters and their values are case sensitive.
    Requests will be ignored while power is off, and while power state is being changed.
    Output is "1" or "0" indicating whether request was accepted, or ignored.
